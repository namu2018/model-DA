1-2 미국에 사는 인디언들의 당뇨병 여부를 예측

Yes or No 문제는 로지스틱 회귀분석(logistic regression)으로 간단하게 풀수 있다.
테스트 데이터 트레이닝(훈련) 데이터를 반으로 쪼개서 
(1) 훈련(트레이닝)데이터로 모델을 만들고
(2) 테스트 데이터로 예측을 하고 평가해 보자.

훈련 데이터, 테스트 데이터 나누기

# install.packages("faraway")
data(pima, package="faraway")
pima$test <- factor(pima$test)
train_idx <- sample(NROW(pima)/2)

모델 만들고 예측하기

glm : 일반화 선형 모델 생성
glm(
  formula,   # 모델 formula
  data,      # 데이터(적용)
  family,    # 연결할 함수 지정, 로지스틱 회귀 모델 경우 "binomial"을 지정. 
  na.action  # 결측치 처리 옵션. na.fail은 결측치가 있으면 오류 메시지 발생. 
)

predict
predict.glm(
   object,    # glm 객체
   newdata,   # 예측을 수행할 대상 데이터 
   # - response : 반응변수(response variable)의 크기로 값을 반환. 
   #   로지스틱 회귀 모델의 경우 p가 여기에 해당
)

로지스틱 회귀 모형 : log(p / (1-p)) = β0 + β1X

ROC Curve를 그리기

ROC 곡선은 분류 모델의 판단결과의 성능을 보여주는 그래프이다.
민감도와 특이도 간의 관계를 표현한 그래프이다. 
일반적으로 민감도와 특이도는 반비례적인 관계를 갖는다. 

ROC 차트(Receiver Operating Characteristic)를 많이 사용.

ROC curve를 그리는 방법
x축을 FP rate로 하고 
y축을 TP rate로 한다. 

[확인 방법]
(1) ROC 곡선이 북서쪽 코너에 가까울 수록 분류 모델의 성능이 우수하다.
(2) 45도 각도의 직선에 다가갈수록 분류 성능이 나쁘다. 
(3) ROC 곡선의 정량적인 성능분석 척도를 AUC(Area Under the Curve)라 함.

AUC(Area Under Curve)는 밑 면적을 의미한다. 넓을 수록 좋은 모델이다.
북서쪽에 가까워질수록 AUC값이 1에 가까워진다. 
1에 가까워질수록 분류 성능이 우수하다. 
AUC값이 0.5에 다가갈수록 분류 성능이 나빠진다.
